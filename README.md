# Claude Code Skills

Personal collection of Claude Code skills.

## Skills

| Skill | Description |
|-------|-------------|
| install-claude-stack | Install Claude Code plugin stack (superpowers + oh-my-claudecode + context7 + serena) |
| context-recovery | Return to main task after completing a side-task |
| pr-dev-to-prod | Auto-generate release PR from dev to prod branch |
| create-issue | GitHub issue creation template |
| create-pr | GitHub PR creation template |
| python-style | Python coding style guide |
| planning-with-files | Task planning and progress tracking (submodule) |
| senior-devops | DevOps/CI/CD automation |
| senior-data-engineer | Data pipeline design |
| senior-backend | Backend API development |
| code-reviewer | Code review checklist |

## Usage

Clone with submodules:

```bash
git clone --recursive https://github.com/devseunggwan/skills.git ~/.claude/skills
```

Update submodules:

```bash
git submodule update --remote
```
